Time Capsule Website
Project Overview

Time Capsule Website is a simple front-end web project developed using HTML, CSS, and JavaScript. This website allows users to write and save a message for their future self. The saved message remains available even after refreshing or reopening the browser using browser storage.

The project is designed to demonstrate basic web development concepts including user input, styling, interactivity, and local storage.

Objective

The purpose of this project is to create a small digital space where users can store thoughts, goals, memories, or messages and view them later.

Features
Write a title and message
Save messages in browser storage
Open and display saved messages
Delete stored messages
Simple and clean interface
Data remains available after refresh
Technologies Used
HTML → Page structure
CSS → Design and styling
JavaScript → Functionality and storage

How It Works
Enter a title.
Write a message.
Click Save Message.
The website stores data using local storage.
Click Open Capsule to view saved content.
Click Delete to remove saved data.
